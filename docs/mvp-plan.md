# Shadow OS MVP Plan

## Product Goal

Build the first usable version of Shadow OS as a zero-cost deployable web app that makes users feel seen, challenged, and guided.

## MVP Outcome

The user should be able to:

1. Sign in
2. Log thoughts
3. Log decisions
4. View AI feedback
5. See simple behavior insights over time

## Best First Build

### Core data types

- User profile
- Thought entry
- Decision entry
- Mood / energy / focus log
- Commitments

### Initial screens

- Landing page
- Login / signup
- Dashboard
- Thought logger
- Decision logger
- Insights page
- Settings / personality mode

## Differentiation Layer

After the MVP works, add:

- Intent vs action gap analysis
- smart behavioral nudges
- life score
- weekly pattern summaries
